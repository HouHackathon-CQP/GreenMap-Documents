    Cài Đặt & Tùy Chỉnh¶

    Truy Cập Cài Đặt¶

    1. Đăng nhập vào tài khoản GreenMap
    2. Nhấp avatar/tên ở góc trên bên phải
    3. Chọn Settings hoặc Cài Đặt

    Hoặc: 1. Vào Profile 2. Nhấp tab Settings

    Cài Đặt Chung¶

    Ngôn Ngữ¶

    Chọn ngôn ngữ hiển thị: - Tiếng Việt (mặc định) - English - Các ngôn ngữ khác (nếu có)

    Múi Giờ¶

    Cài đặt múi giờ để hiển thị thời gian chính xác: - UTC+7 (Việt Nam, mặc định) - Các múi giờ khác

    Định Dạng Ngày Tháng¶

    Chọn cách hiển thị ngày: - DD/MM/YYYY (25/12/2024) - MM/DD/YYYY (12/25/2024) - YYYY-MM-DD (2024-12-25)

    Đơn Vị Đo Lường¶

    Chọn hệ thống đơn vị: - Metric (km, °C, mặc định) - Imperial (miles, °F)

    Cài Đặt Bản Đồ¶

    Lớp Bản Đồ Mặc Định¶

    Chọn lớp hiển thị khi mở ứng dụng: - OpenStreetMap (mặc định) - Satellite - Terrain

    Lớp Dữ Liệu Mặc Định¶

    Chọn lớp dữ liệu hiển thị: - ☑️ Sensors - Cảm biến - ☑️ Bicycle Rental - Trạm xe đạp - ☑️ Charging Stations - Trạm sạc - ☑️ Parks - Công viên - ☑️ Tourist Attractions - Điểm du lịch

    Zoom Level¶

    Cài đặt mức zoom mặc định: - Min Zoom: Tối thiểu (1-5) - Max Zoom: Tối đa (15-20)

    Animation¶

    Bật/tắt các hiệu ứng: - ☑️ Pan Animation - Chuyển động khi pan - ☑️ Zoom Animation - Chuyển động khi zoom - ☑️ Marker Animation - Hiệu ứng marker

    Cài Đặt Thông Báo¶

    Cảnh Báo AQI¶

    Bật/tắt cảnh báo chất lượng không khí:

    • ☑️ Enable AQI Alerts - Bật cảnh báo
    • Threshold Level: Mức cảnh báo (0-500)
    • Alert Type:
    • ☑️ Browser Notification
    • ☑️ Email Notification
    • ☑️ In-App Alert

    Cập Nhật Báo Cáo¶

    Thông báo khi: - ☑️ New Comment - Có bình luận mới - ☑️ Status Changed - Trạng thái thay đổi - ☑️ Resolved - Báo cáo được giải quyết - Delivery: Email, In-App, hoặc cả hai

    Tin Tức & Cộng Đồng¶

    • ☑️ Community News - Tin tức từ cộng đồng
    • ☑️ System Updates - Cập nhật hệ thống
    • ☑️ Event Notifications - Thông báo sự kiện

    Tần Suất Thông Báo¶

    Chọn tần suất nhận email: - Real-time - Ngay khi có - Daily - Mỗi ngày - Weekly - Mỗi tuần - Never - Không bao giờ

    Cài Đặt Quyền Riêng Tư¶

    Mức Độ Công Khai Profile¶

    • Public: Bất cứ ai cũng có thể xem profile
    • Friends Only: Chỉ bạn bè có thể xem
    • Private: Chỉ bạn có thể xem

    Chia Sẻ Vị Trí¶

    • ☑️ Share Current Location - Chia sẻ vị trí hiện tại
    • Khi bật: Người khác sẽ thấy bạn đang ở đâu
    • Khi tắt: Vị trí bạn bị ẩn

    Tên Công Khai Trong Báo Cáo¶

    • ☑️ Show My Name - Hiển thị tên bạn
    • ☑️ Anonymous - Ẩn danh (mặc định)

    Rõ Ràng Hoạt Động¶

    • ☑️ Make Activity Visible - Người khác thấy hoạt động của bạn
    • ☑️ Private Activity - Hoạt động chỉ riêng bạn

    Cài Đặt Bảo Mật¶

    Quản Lý Phiên Làm Việc¶

    Xem các thiết bị đã đăng nhập: - Current Device: Thiết bị hiện tại - Other Devices: Các thiết bị khác - Last Activity: Hoạt động lần cuối

    Bạn có thể: - Xem chi tiết thiết bị - Đăng xuất thiết bị từ xa - Xóa phiên làm việc cũ

    Đăng Xuất Từ Tất Cả Thiết Bị¶

    Nhấp Sign Out All Devices để: - Đăng xuất khỏi tất cả thiết bị - Buộc phải đăng nhập lại trên tất cả

    Yêu Cầu Xác Nhận¶

    • ☑️ Two-Factor Authentication - Xác thực 2 bước (nếu có)
    • ☑️ Email Verification - Xác minh email cho các hành động nhạy cảm

    Cài Đặt Dữ Liệu & Lưu Trữ¶

    Cache & Bộ Nhớ Tạm¶

    • Clear Cache: Xóa dữ liệu bộ nhớ tạm
    • Clear Cookies: Xóa cookies
    • Clear All Data: Xóa tất cả dữ liệu (cảnh báo!)

    Export Dữ Liệu¶

    Xuất dữ liệu của bạn: 1. Nhấp Export My Data 2. Chọn định dạng: - JSON: Dữ liệu cấu trúc - CSV: Bảng tính - PDF: Tài liệu 3. Download file

    Import Dữ Liệu¶

    Nhập dữ liệu từ file: 1. Nhấp Import Data 2. Chọn file 3. Chọn lựa chọn merge 4. Confirm import

    Cài Đặt Nâng Cao¶

    API Keys (Cho Developers)¶

    Nếu bạn là developer, bạn có thể: 1. Tạo API key mới 2. Xem các keys hiện tại 3. Xóa/regenerate keys 4. Đặt quyền (scopes)

    Webhook (Tùy Chọn)¶

    Nếu có sẵn: - Tạo webhooks cho events - Quản lý URLs - Xem history webhook calls

    Các Tùy Chọn Khác¶

    Chế Độ Tiết Kiệm¶

    • ☑️ Data Saver Mode - Giảm dữ liệu dùng
    • ☑️ Lite Mode - Giao diện nhẹ hơn

    Thông Báo Cập Nhật¶

    • ☑️ Auto-Update - Tự động cập nhật khi có phiên bản mới
    • ☑️ Beta Features - Sử dụng tính năng beta

    Cần giúp? Xem User Guide Overview hoặc liên hệ support.
