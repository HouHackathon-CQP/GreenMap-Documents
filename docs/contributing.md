# Đóng Góp

Chúng tôi rất hoan nghênh mọi đóng góp từ cộng đồng!

## Quy Trình Đóng Góp

1. **Fork** repository mà bạn muốn đóng góp.
2. **Clone** về máy của bạn.
3. Tạo một **branch** mới cho tính năng hoặc bản sửa lỗi của bạn (`git checkout -b feature/amazing-feature`).
4. **Commit** các thay đổi của bạn (`git commit -m 'Add some amazing feature'`).
5. **Push** lên branch (`git push origin feature/amazing-feature`).
6. Tạo một **Pull Request**.

## Quy Tắc Code (Code of Conduct)

Xin vui lòng đọc kỹ [CODE_OF_CONDUCT.md](https://github.com/HouHackathon-CQP/.github/blob/main/CODE_OF_CONDUCT.md) trước khi tham gia.
